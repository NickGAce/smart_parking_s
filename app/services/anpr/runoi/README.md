# Runoi model directory

Put model files here (or set custom env vars):
- `models/yolo/best.pt`  ← copy from `Runoi/ANPR-System/models/yolo/model/best.pt`
- `models/crnn/crnn_ocr_model_int8_fx.pth` ← copy from `Runoi/ANPR-System/models/ocr_crnn/quant/crnn_ocr_model_int8_fx.pth`

Helper:
```bash
./scripts/setup_runoi_models.sh /path/to/ANPR-System
```
